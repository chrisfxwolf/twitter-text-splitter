Quick Start

Installation

git clone https://github.com/yourusername/twitter-text-splitter.git
cd twitter-text-splitter
chmod +x twitter_split.sh

Basic Usage

# From file
./twitter_split.sh -f your_text.txt

# From standard input
echo "Your long text here" | ./twitter_split.sh
